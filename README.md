# 📂 Smart File Organizer (Python)

A simple yet powerful Python tool that automatically organizes files into folders based on their type (e.g., Images, Documents, Videos, etc.).

---

## 🚀 Features

- 📁 Automatically sorts files into categories
- ⚡ Fast and efficient file handling
- 🧠 Smart file type detection
- 🛠 Easy to customize
- 💻 Beginner-friendly code

---

## 📌 How It Works

The script scans a selected directory and moves files into specific folders based on their extensions:

| File Type | Folder |
|----------|--------|
| .jpg, .png | Images |
| .pdf, .docx | Documents |
| .mp4, .mkv | Videos |
| .mp3 | Audio |
| Others | Misc |

---

## 🧰 Technologies Used

- Python 3
- OS module
- Shutil module
